# VOLL Pilates - Login Page

## Descrição
Página de login simples criada para demonstração, utilizando Tailwind CSS.

### Recursos
- Design responsivo
- Estilização moderna
- Login fake com alerta

### Como usar
Abra o `index.html` no seu navegador.

### Configuração do GitHub Pages

Para publicar no GitHub Pages:
1. Vá em "Settings" do repositório
2. Clique em "Pages" no menu lateral
3. Em "Source", selecione a branch "main"
4. Na pasta, selecione "/root"
5. Salve as configurações

Após isso, seu site estará disponível em: 
https://vollpilates.github.io/voll-login-page/
